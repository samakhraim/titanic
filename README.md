# Titanic Survival Prediction & Exploratory Data Analysis

## Overview

This project explores the Titanic dataset from Kaggle using Python for data cleaning, exploratory data analysis (EDA), and visualization. The goal is to identify factors that influenced passenger survival and gain practical experience in data analysis workflows.

## Dataset

Source:
https://www.kaggle.com/competitions/titanic

The dataset contains passenger information including:

* Age
* Sex
* Passenger Class
* Fare
* Embarked Port
* Family Information
* Survival Status

## Objectives

* Perform data cleaning and preprocessing
* Analyze missing values and data quality
* Explore relationships between features and survival
* Create meaningful visualizations
* Generate insights from historical passenger data

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Exploratory Data Analysis

The analysis includes:

* Missing value treatment
* Distribution analysis
* Correlation analysis
* Survival rate comparisons
* Feature relationship visualization

## Key Findings

* Female passengers had significantly higher survival rates.
* Passenger class strongly influenced survival probability.
* Younger passengers generally had better survival outcomes.
* Fare and socioeconomic status showed a relationship with survival.

## Learning Outcomes

Through this project, I developed practical experience in:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization
* Statistical reasoning
* Insight generation from real-world datasets
